# Rule-Based Chatbot Web

This is a simple Rule-Based Chatbot developed using Python and Flask for the CodSoft AI Internship.

## Author

Shreyas Mohapatra

## Features

* Greeting responses
* Emotion detection
* Date and time
* Basic questions and answers
* Random jokes
* Simple web interface

## Requirements
* Python 3.14.6
* Flask  3.1.3

## How to Run
Install Flask:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python app.py
```

Open your browser:

```text
http://127.0.0.1:5000
```
